<!-- INSTRUÇÕES PARA ADICIONAR A FOTO DE PERFIL -->

1. PARA ADICIONAR A TUA FOTO:
   - Coloca a tua foto (ex: "perfil.jpg" ou "perfil.png") na pasta: src/assets/
   - Depois edita o ficheiro tab3.page.ts
   - Na linha onde diz: foto: 'assets/perfil.png',
   - Substitui "perfil.png" pelo nome do teu ficheiro

2. EXEMPLO:
   Se o teu ficheiro se chama "minha-foto.jpg":
   foto: 'assets/minha-foto.jpg',

3. FORMATOS ACEITES:
   - .jpg / .jpeg
   - .png
   - .gif
   - .webp

4. TAMANHO RECOMENDADO:
   - Quadrada (ex: 400x400px ou 500x500px)
   - Menos de 1MB

5. ALTERNATIVA - USAR URL ONLINE:
   Se preferires usar uma foto online (ex: do Instagram), podes usar:
   foto: 'https://www.exemplo.com/minha-foto.jpg',
